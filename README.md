angular-maha-bower
==================

Usage
-----

Just include countries.js and proper language file

```html
<script src="/angular-maha-countries/countries.js"
type="text/javascript"></script>
<script src="/angular-maha-countries/en.js"
type="text/javascript"></script>
```
